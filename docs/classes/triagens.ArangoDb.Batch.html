      __construct(\triagens\ArangoDb\Connection $connection, $options) : \triagens\ArangoDb\Batch

      Batch instance by default starts capturing request after initiated. To disable this, pass startCapture=>false inside the options array parameter

      Parameters

      $connection

      \triagens\ArangoDb\Connection

      that this batch class will monitor for requests in order to batch them. Connection parameter is mandatory.

      $options

      An array of options for Batch construction. See below for options:

      Options are :

    • 'sanitize' - True to remove _id and _rev attributes from result documents returned from this batch. Defaults to false.
    • '$startCapture' - Start batch capturing immediately after batch instantiation. Defaults to true.
