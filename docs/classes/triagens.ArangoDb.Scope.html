      Destroy the scope

      __destruct() : void

      This will call leave() to ensure the scope is definitely left

      Leave the scope

      leave() : void

      This will execute the exit func with the value returned by the call to the init func. leave() will check whether the scope has already been left to avoid duplicate execution of the exit func.
